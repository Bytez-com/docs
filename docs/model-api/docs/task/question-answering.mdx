<Accordion defaultOpen title="Basic usage">
  Send a question with context to a model to generate an answer:

  <CodeGroup>
    ```javascript javascript
    import Bytez from 'bytez.js'

    const sdk = new Bytez("BYTEZ_KEY");
    const model = sdk.model("cloudqi/deepset/roberta-base-squad2");

    await model.create()

    const { error, output } = await model.run({
      question: "Where does Holly live?",
      context: "My name is Holly and I live in NYC"
    });

    console.log({ error, output })
    ```
    ```python python
    from bytez import Bytez

    sdk = Bytez("BYTEZ_KEY")
    model = sdk.model("cloudqi/deepset/roberta-base-squad2")

    model.create()

    output, error = model.run({
      "question": "Where does Holly live?",
      "context": "My name is Holly and I live in NYC"
    })

    print({ "error": error, "output": output })
    ```
    ```bash http
    curl -X POST 'https://api.bytez.com/models/v2/cloudqi/deepset/roberta-base-squad2' \
    -H 'Authorization: Key BYTEZ_KEY' \
    -H 'Content-Type: application/json' \
    --data '{
      "question": "Where does Holly live?",
      "context": "My name is Holly and I live in NYC"
    }'
    ```
