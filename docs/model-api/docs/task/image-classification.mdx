<Accordion defaultOpen title="Basic usage">
  Send an image to a model to generate classification labels.

  <CodeGroup>
    ```javascript javascript
    import Bytez from 'bytez.js'

    const sdk = new Bytez("BYTEZ_KEY");
    const model = sdk.model("google/vit-base-patch16-224");

    await model.create()

    const image = "https://www.padoniavets.com/sites/default/files/field/image/cats-and-dogs.jpg"
    const { error, output } = await model.run({ url: image });

    console.log({ error, output })
    ```
    ```python python
    from bytez import Bytez

    sdk = Bytez("BYTEZ_KEY")
    model = sdk.model("google/vit-base-patch16-224")

    model.create()

    image = "https://www.padoniavets.com/sites/default/files/field/image/cats-and-dogs.jpg"
    output, error = model.run({ "url": image })

    print({ "error": error, "output": output })
    ```
    ```bash http
    curl -X POST 'https://api.bytez.com/models/v2/google/vit-base-patch16-224' \
    -H 'Authorization: Key BYTEZ_KEY' \
    -H 'Content-Type: application/json' \
    --data '{
      "url": "https://www.padoniavets.com/sites/default/files/field/image/cats-and-dogs.jpg"
    }'
    ```
