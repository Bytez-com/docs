<Accordion defaultOpen title="Basic usage">
  Send a text input and a set of candidate labels to receive classification results:

  <CodeGroup>
    ```javascript javascript
    import Bytez from 'bytez.js'

    const sdk = new Bytez("BYTEZ_KEY");
    const model = sdk.model("facebook/bart-large-mnli");

    await model.create()

    const { error, output } = await model.run({
      text: "One day I will see the world",
      candidate_labels: ["travel", "cooking", "dancing"]
    });

    console.log({ error, output })
    ```
    ```python python
    from bytez import Bytez

    sdk = Bytez("BYTEZ_KEY")
    model = sdk.model("facebook/bart-large-mnli")

    model.create()

    output, error = model.run({
      "text": "One day I will see the world",
      "candidate_labels": ["travel", "cooking", "dancing"]
    })

    print({ "error": error, "output": output })
    ```
    ```bash http
    curl -X POST 'https://api.bytez.com/models/v2/facebook/bart-large-mnli' \
    -H 'Authorization: Key BYTEZ_KEY' \
    -H 'Content-Type: application/json' \
    --data '{
      "text": "One day I will see the world",
      "candidate_labels": ["travel", "cooking", "dancing"]
    }'
    ```
