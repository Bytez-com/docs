<Accordion defaultOpen title="Basic usage">
  Send a text input to classify individual tokens:

  <CodeGroup>
    ```javascript javascript
    import Bytez from 'bytez.js'

    const sdk = new Bytez("BYTEZ_KEY");
    const model = sdk.model("dslim/bert-base-NER");

    await model.create()

    const { error, output } = await model.run("John Doe is a software engineer at Google");

    console.log({ error, output })
    ```
    ```python python
    from bytez import Bytez

    sdk = Bytez("BYTEZ_KEY")
    model = sdk.model("dslim/bert-base-NER")

    model.create()

    output, error = model.run("John Doe is a software engineer at Google")

    print({ "error": error, "output": output })
    ```
    ```bash http
    curl -X POST 'https://api.bytez.com/models/v2/dslim/bert-base-NER' \
    -H 'Authorization: Key BYTEZ_KEY' \
    -H 'Content-Type: application/json' \
    --data '{
      "text": "John Doe is a software engineer at Google"
    }'
    ```
