<Accordion defaultOpen title="Basic usage">
  Send a video input URL to classify its contents:

  <CodeGroup>
    ```javascript javascript
    import Bytez from 'bytez.js'

    const sdk = new Bytez("BYTEZ_KEY");
    const model = sdk.model("ahmedabdo/video-classifier");

    await model.create()

    const video = "https://video-previews.elements.envatousercontent.com/6d07b79d-b17a-47b5-9d24-4fe984c7ca36/watermarked_preview/watermarked_preview.mp4"
    const { error, output } = await model.run({ url: video });

    console.log({ error, output })
    ```
    ```python python
    from bytez import Bytez

    sdk = Bytez("BYTEZ_KEY")
    model = sdk.model("ahmedabdo/video-classifier")

    model.create()

    video = "https://video-previews.elements.envatousercontent.com/6d07b79d-b17a-47b5-9d24-4fe984c7ca36/watermarked_preview/watermarked_preview.mp4"
    output, error = model.run({ "url": video })

    print({ "error": error, "output": output })
    ```
    ```bash http
    curl -X POST 'https://api.bytez.com/models/v2/ahmedabdo/video-classifier' \
    -H 'Authorization: Key BYTEZ_KEY' \
    -H 'Content-Type: application/json' \
    --data '{
      "url": "https://video-previews.elements.envatousercontent.com/6d07b79d-b17a-47b5-9d24-4fe984c7ca36/watermarked_preview/watermarked_preview.mp4"
    }'
    ```
