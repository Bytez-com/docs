## Quickstart
### Extract Features from Text
Send text to a model to generate embeddings.

<CodeGroup>
```javascript javascript 
import Bytez from "bytez.js";

const client = new Bytez("YOUR_BYTEZ_KEY_HERE");
const model = client.model("Salesforce/SFR-Embedding-2_R");

const inputText = "Your text for feature extraction goes here...";

const { error, output } = await model.run(inputText);

if (error) {
  console.error("Error:", error);
} else {
  console.log("Embedding:", output[0]);
}
```
