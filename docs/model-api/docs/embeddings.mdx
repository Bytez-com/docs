## Quickstart 
### Generate Text Embeddings

Send a text input to generate an embedding.
<CodeGroup>

```javascript javascript
import Bytez from "bytez.js";

const client = new Bytez("YOUR_BYTEZ_KEY_HERE");
const model = client.model("nomic-ai/nomic-embed-text-v1.5");

const input = "Once upon a time";
const { error, output } = await model.run(input);

if (error) {
  console.error("Error:", error);
} else {
  console.log(output);
}
```
```python python
from bytez import Bytez

client = Bytez("API_KEY")

# 1) Select the model
model = client.model("nomic-ai/nomic-embed-text-v1.5")

# 2) Load the model
model.load()

# 3) Run the model
output = model.run("Once upon a time")

print(output)
```
```julia julia
using Bytez

client = Bytez.init("API_KEY");

# 1) Select the model
model = client.model("nomic-ai/nomic-embed-text-v1.5")

# 2) Load the model
model.load()

# 3) Run the model
output = model.run("Once upon a time")

println(output)
```
```bash bash
curl --location 'https://api.bytez.com/models/v2/nomic-ai/nomic-embed-text-v1.5' \
--header 'Authorization: Key API_KEY' \
--header 'Content-Type: application/json' \
--data '{
    "text": "Once upon a time"
}'
```
</CodeGroup>

## Generate Vector Embeddings
Extract feature representations for text using vector embeddings.

<CodeGroup>

```javascript javascript
import Bytez from "bytez.js";

const client = new Bytez("YOUR_BYTEZ_KEY_HERE");
const model = client.model("mixedbread-ai/mxbai-embed-large-v1");

const input = "Analyzing feature embeddings";
const { error, output } = await model.run(input);

if (error) {
  console.error("Error:", error);
} else {
  console.log(output);
}
```
